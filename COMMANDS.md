## 📖 HOW TO USE

### Basic Usage:
```
.command
```

### With Arguments:
```
.command argument1 argument2
```

### Reply to Message:
```
Reply to a message and type: .command
```

### Example:
```
.song Despacito
.sticker (reply to photo)
.tagall Hello everyone!
```

## 📝 NOTES

1. **Prefix:** සියලු commands `.` (dot) එකෙන් start වෙනවා
2. **Admin Commands:** Group admin විතරක් use කරන්න පුළුවන්
3. **Owner Commands:** Bot owner විතරක් use කරන්න පුළුවන්
4. **Reply Commands:** කිහිපයක් messages වලට reply කරන්න ඕනි
5. **Arguments:** [] දාන්න ඕනි නැහැ, example විතරයි
